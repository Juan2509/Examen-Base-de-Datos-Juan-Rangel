# Examen Base de Datos Juan Rangel (Campus Pizza)



### Objetivo

Este examen consistía en el desarrollo de una base de datos para una pizzería con el fin de mejorar la gestión de una pizzería mediante el uso de MySQL. La base de datos debía contener tablas que almacenaran información como productos, los clientes, ingredientes, entre otros.

### Estructura

La estructura del proyecto contiene el archivo SQL, el cuál tendrá el trabajo hecho desde MySQL Workbench y las capturas con el esquema de la base de datos plasmada en DrawSQL y directamente en MySQL Workbench.

### Funcionamiento

Lastimosamente este proyecto no fue exitoso debido a varias dificultades durante la ejecución del código como errores que no pudieron ser identificados los cuales no permitían la ejecución correcta del flujo y además durante el desarrollo no hubo una idea clara de como estructurar las tablas y su contenido, sin embargo se pueden visualizar algunas tablas y su contenido con el comando "SELECT * FROM (la tabla que deseas ver);".

Si se desea probar el archivo SQL será necesario abrir MySQL Workbench y abrir el archivo, una vez allí solo será necesario seleccionar la opción file y posteriormente seleccionar open SQL Script y finalmente ahí se podrá visualizar el código que contiene la base de datos junto con sus respectivas tablas.
